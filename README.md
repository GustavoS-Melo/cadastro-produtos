# Cadastro de Produtos — Projeto de Portfólio

Este projeto consiste em uma aplicação de cadastro e gerenciamento de produtos, desenvolvida com foco em estudo e portfólio, simulando um cenário comum em sistemas administrativos e aplicações web.

O objetivo principal foi praticar lógica de programação, manipulação de dados e organização de interface no front-end.

## Sobre o projeto
A aplicação permite cadastrar, visualizar e gerenciar produtos por meio de uma interface simples e funcional. Durante o desenvolvimento, o foco esteve em criar uma estrutura clara, fácil de manter e com boa experiência de uso.

Este projeto representa um exercício prático de funcionalidades frequentemente utilizadas em sistemas reais, como painéis administrativos e ferramentas internas.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos cadastrados
- Interação dinâmica com os dados
- Interface responsiva

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Git e GitHub

## Deploy

## Objetivo do projeto
- Praticar lógica de programação em JavaScript
- Trabalhar manipulação e organização de dados
- Simular um sistema simples de gerenciamento de produtos
- Desenvolver um projeto funcional para portfólio

## Observações
Este projeto não possui integração com banco de dados ou autenticação, sendo utilizado exclusivamente para fins educacionais e demonstrativos.
